# OperatingSystem

这是南开大学操作系统课程大作业小组提交的 ucore 操作系统代码仓库。

## ucore 操作系统实验框架

### 项目简介
本项目基于 RISC-V 指令集架构的轻量级操作系统 ucore 改编而来，ucore 源自 MIT 的 xv6 操作系统与清华大学的 ucore 实验框架，核心目标是通过补全操作系统关键模块的实现，帮助学习者深入理解操作系统的核心原理与底层实现逻辑。本项目作为操作系统课程大作业，基于课程提供的 ucore 实验框架（框架中预留了页表管理、进程调度、中断异常处理、系统调用、物理内存分配等核心模块的空缺），完成了上述关键模块的代码补全与功能验证，最终实现了一个可在 RISC-V 架构下正常运行的轻量级操作系统。
